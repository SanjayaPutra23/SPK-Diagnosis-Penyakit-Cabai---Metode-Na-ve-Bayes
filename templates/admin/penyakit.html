1. Apa itu Nilai Prior P(H)?

Nilai {{ penyakit.prior_probability }} (atau {{ (penyakit.prior_probability * 100)|round }}%) adalah nilai Prior Probability. Ini merepresentasikan probabilitas awal kemunculan penyakit ini di lahan pertanian secara umum, sebelum adanya bukti gejala apapun yang diamati.

2. Dari mana angka ini berasal?

Nilai ini didapatkan dari proses perhitungan statistik berdasarkan basis data pakar:

Rumus: P(H) = (Jumlah kemunculan Penyakit X) / (Total seluruh kejadian penyakit)

P({{ penyakit.kode_penyakit }}) = {{ penyakit.prior_probability }}
Syarat Naïve Bayes: Total seluruh nilai P(H) untuk semua penyakit dalam sistem ini jika dijumlahkan harus bernilai tepat 1.0000 (100%).
